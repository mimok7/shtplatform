# 바야 소울 크루즈 데이터 입력 지침 (Bhaya Soul Cruise)

## 1. 크루즈 기본 정보
- **크루즈명**: 바야 소울 크루즈 (Bhaya Soul Cruise)
- **일정 타입**: 1N2D (1박 2일 기준)
- **통화**: VND (베트남 동)
- **유효 연도**: 2026

## 2. 객실 목록 및 특징
- **소울 레거시 스위트 (Soul Legacy Suite)**
  - 80m2, 2층, 슈퍼 킹베드 
  - 기본 성인 2인
  - 커넥팅룸 없음
  - 발코니, 객실 내 욕조, 스탠딩 샤워 제공
  - 엑스트라 베드 추가 가능

## 3. 가격 정책 규칙
- **성인 (Adult)**: 1인 단가
- **아동 (Child)**: 6~12세 아동 단가 적용. 단, **성인 1인 + 아동 1인 투숙 시 아동도 성인요금** 적용.
- **엑스트라 베드**: 성인 2인 + 아동 1인이 엑스트라 베드 사용 시 부과되는 요금.
- **싱글차지**: 성인 1인이 객실을 단독 사용할 때의 총 요금.
- **투숙 불가**: 성인 2인 + 아동 2인으로는 1개의 객실(소울 레거시 스위트) 사용 불가.

## 4. 요금 포함/불포함 사항
- **포함 사항**: 웰컴드링크, 외부투어 입장료/관광비용, 객실 숙박, 총 3회의 식사, 해피아워 까나페, 수영장, 40분 허브족욕 및 무료 웰니스 트리트먼트 서비스.
- **불포함 사항**: 항공권, 왕복 교통편, 비자, 마사지 서비스, 음료/주류, 개인경비 등. (단, 서호 지역 셔틀차량 픽업/드랍 요청 시 편도 팀당 50만동 추가 비용 발생).

## 5. 변경 이력
- 2026-07-01: 신규 크루즈 가이드라인 작성 및 2026년 요금 등록 완료.
